# Repository Instructions

- All commits in this repository must be signed. Do not create unsigned commits.
- Git commit signing in this repo uses the 1Password SSH signing agent via `op-ssh-sign`.
- If signing fails in a sandboxed environment, use the local 1Password agent socket at `$HOME/Library/Group Containers/2BUA8C4S2C.com.1password/t/agent.sock` and request the necessary approval rather than bypassing signing.
